# Phase 1 Clean Room Record

Purpose: keep a clear project record that Persistence Gate is developed as an independent, general-purpose project using public or synthetic material.

## Project boundaries

Use:

- personal project accounts
- public documentation
- synthetic examples
- project-owned benchmark data

Avoid:

- employer documents
- employer code
- employer screenshots
- private tickets or emails
- customer/member/claim data
- production system exports

## Working rules

- Keep product code in this repo.
- Keep benchmarks reproducible with public or synthetic sources.
- Keep dated notes for major design decisions.
- Do not connect this project to private systems unless there is a formal approved pilot.

## Current project description

Persistence Gate is an evidence-admission layer for AI/RAG systems. It sits between retrieval and generation, then decides which retrieved evidence may influence the downstream answer.
